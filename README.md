# 催眠控制系统 · Hypnosis Control System

> 仿日本动漫风格的催眠App模拟器，用于剧情扮演/故事创作场景的道具。纯前端实现，无需后端服务。

## 使用说明

输入任意用户名和符合格式的访问密钥即可登录。

### 密码格式

```
[可选前缀] + ly + 套餐等级(2位) + 使用等级(2位) + 经验值(3位) + 本月使用次数(4位) + 错误详情开关(1位) + [可选后缀]
```

密码中只要包含 `ly` 即可，`ly` 前后可以有任意额外字符。系统会自动定位 `ly` 并从该位置开始解析。

**密码格式错误时统一提示"密码错误"，不会暴露具体错误原因。**

| 字段 | 位置 | 位数 | 范围 | 说明 |
|------|------|------|------|------|
| 可选前缀 | - | 任意 | 任意字符 | `ly` 之前的字符会被忽略 |
| 固定标识 | - | 2 | `ly` | 密码中必须包含 `ly` |
| 套餐等级 | 3-4 | 2 | 00-99 | 决定套餐类型和每月最大使用次数 |
| 使用等级 | 5-6 | 2 | 00-99 | 用户的使用等级（Lv.） |
| 经验值 | 7-9 | 3 | 000-999 | 当前使用等级的经验值 |
| 使用次数 | 10-13 | 4 | 0000-9999 | 本月已使用次数 |
| 详情开关 | 14 | 1 | 0-9 | 0=简略错误提示，其他=详细错误提示 |
| 可选后缀 | 15+ | 任意 | 任意字符 | 会被忽略 |

### 套餐等级对照表

| 等级范围 | 套餐名称 | 月最大使用次数 | 图标 |
|---------|---------|-------------|------|
| 0 | 体验用户 | 5 | 🌱 |
| 1 - 3 | 初级套餐 | 20 | 🛡 |
| 4 - 6 | 标准套餐 | 50 | ⭐ |
| 7 - 9 | 进阶套餐 | 100 | ⚡ |
| 10 - 14 | 高级套餐 | 200 | 💎 |
| 15 - 19 | 超级套餐 | 500 | 🔥 |
| 20+ | 至尊套餐 | 9999 | 👑 |

### 使用等级升级

每个使用等级升级所需经验 = `(当前等级 + 1) × 100`

例如：Lv.5 升到 Lv.6 需要 600 经验

### 密码示例

| 密码 | 套餐 | 使用等级 | 经验 | 使用次数 | 详情开关 | 效果 |
|------|------|---------|------|---------|---------|------|
| `ly031512000101` | 初级(Lv.3) | Lv.15 | 120 | 10 | 开 | 正常使用 |
| `ly070808000051` | 进阶(Lv.7) | Lv.8 | 80 | 5 | 开 | 正常使用 |
| `ly200099900011` | 至尊(Lv.20) | Lv.0 | 999 | 1 | 开 | 至尊套餐 |
| `ly000005000001` | 体验(Lv.0) | Lv.0 | 50 | 0 | 开 | 体验用户 |
| `ly030015002510` | 初级(Lv.3) | Lv.0 | 150 | 25 | 关 | 超限(25>20)，按钮禁用 |

### 错误详情开关

- **开关 = 0**：无论什么错误，统一提示"密码错误"
- **开关 = 1-9**：显示具体的错误原因（如"套餐等级编码异常"、"经验数据异常"等）
- 当密码格式不完整、无法提取开关值时，默认按详细模式显示

## 控制模式

应用提供四种控制模式，每种模式有独特的动画效果：

| 模式 | 动画效果 |
|------|---------|
| 催眠 | 旋转的阿基米德螺旋线 |
| 发情 | 跳动的爱心 + 外扩的心形波纹 |
| 拘束 | 从中心外扩的同心圆 |
| 常识改变 | 文字雨 + 脉动的脑图标 + 故障闪烁 |

## 技术栈

- 纯 HTML5 / CSS3 / ES6+ JavaScript
- 无框架、无构建工具、无后端
- Google Fonts (Inter) + Font Awesome 6.5
- 移动端响应式设计
